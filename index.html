Сервис прямых ссылок

Этот сервер используется для временного хранения файлов, загруженных через Video Downloader Bot.

Как это работает

Когда вы запрашиваете загрузку видео через Telegram-бот, и файл оказывается слишком большим для отправки в Telegram (>50 МБ), бот может создать временную прямую ссылку для скачивания.

Ссылки действительны в течение 24 часов, после чего файлы автоматически удаляются.

Важно!

Прямые ссылки для скачивания можно получить только через Telegram-бот. Этот сайт не предоставляет функционал для загрузки файлов напрямую.
